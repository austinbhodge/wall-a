# Chassis — 3D Printed Tank

Based on [Thingiverse thing:2753227](https://www.thingiverse.com/thing:2753227) — a small tracked robot for prototyping using 22mm skateboard bearings.

## Print Settings

| Setting | Value |
|---------|-------|
| Material | PLA or PETG |
| Layer Height | 0.2mm |
| Infill | 20-30% |
| Supports | Yes (for sprocket/idler bearing seats) |
| Printer | Bambu Lab A1 Mini (120mm+ bed) |

## Assembly

- 8x 22mm skateboard bearings (8mm bore, 7mm thick)
- 24x M3 screws (10mm)
- 28x M3 nuts
- 8x M3 standoffs (30mm)
- 1x Tamiya 70097 Twin-Motor Gearbox Kit (set to **203:1 gear ratio**)
